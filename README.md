# opthjama

This repository contains code-only research scripts exported from the local `NHANES数据/analysis_ascii` workspace.

Included:

- NHANES 2005-2008 diabetic retinopathy analysis scripts
- ED boarding study analysis and reporting scripts
- Manuscript-facing notes and helper scripts

Excluded intentionally:

- Raw data
- Intermediate analysis outputs
- Large generated tables and figures
- Word reports and other binary artifacts

The upload is organized to preserve the original local project structure as closely as possible.
